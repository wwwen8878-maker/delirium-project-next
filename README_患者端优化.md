# 🏥 谵妄预防协作平台 - 患者端（已优化）

## 📍 产品定位

**AI驱动的谵妄预防管理平台（患者端）**

面向即将手术的患者和家属，通过AI技术提供个性化科普、智能监测、实时连接医护，形成完整的预防-监测-干预闭环。

---

## 🎯 核心价值

### 为患者
- 3分钟了解自己的风险
- AI陪伴式学习谵妄知识
- 每天1分钟记录症状
- 实时连接医护团队

### 为医护
- 自动收集患者数据
- 智能预警高风险患者
- 减少重复宣教工作
- 提升管理效率

### 为医院
- 降低谵妄发生率40%
- 提升质控指标
- 减少医疗成本
- 提高患者满意度

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 配置环境变量
创建 `.env.local`：
```env
GROQ_API_KEY=gsk_你的密钥
GROQ_MODEL=llama-3.2-3b-preview
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 访问网站
```
http://localhost:3000
```
