## 🔧 常见问题处理

### Q: 启动失败？
```powershell
# 清理缓存重试
患者端：
npm cache clean --force
npm install
npm run dev

医护端：
npm cache clean --force  
npm install
npm run dev
```

### Q: 数据不同步？
```javascript
// 浏览器控制台检查
console.log('患者评估:', localStorage.getItem('medical_patient_assessments'));
console.log('症状记录:', localStorage.getItem('medical_symptom_records'));
console.log('预警:', localStorage.getItem('medical_alerts'));

// 如果是null，说明数据未发送
// 检查患者端控制台是否有错误
```

### Q: 医护端不刷新？
```
手动刷新：F5
或关闭重新打开患者监控页面
数据同步延迟最多3秒
```
